# Flask-SSE demo


## Online Demo
https://helloflask.pythonanywhere.com/sse

## Installation

```
$ git clone https://github.com/helloflask/flask-sse-demo
$ cd flask-sse-demo
$ pipenv install
```

## Run

Install and run Redis server:
```
$ redis-server
```

Just excute:
```
$ flask run
```
Then go to http://localhost:5000
